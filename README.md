# Contractor Management App

A comprehensive workforce management solution designed for contractors and construction companies to efficiently manage workers, track attendance, and generate detailed reports.

## About

This application helps contractors and construction companies streamline their daily operations by providing tools for:

- **Worker Management** - Add, edit, and manage worker profiles with complete details
- **Attendance Tracking** - Mark daily attendance with full day, half day, and absent options
- **Holiday Management** - Configure holidays and manage leave schedules
- **Reports & Analytics** - Generate detailed reports for payroll calculation and performance tracking
- **Dashboard** - Get a quick overview of workforce status and attendance summary

## Features

- Phone OTP based authentication (Login/Signup)
- Bilingual support (English/Hindi)
- Responsive design for mobile and desktop
- Dark/Light theme support
- Offline-capable with local storage
- Clean and intuitive user interface

## Tech Stack

- **Frontend**: React 18 with Vite
- **UI Library**: Material-UI (MUI)
- **Routing**: React Router v6
- **State Management**: React Context API
- **Date Handling**: date-fns with MUI Date Pickers
- **Build Tool**: Vite

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/vishalsoni/contractor-app.git

# Navigate to project directory
cd contractor-app

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
